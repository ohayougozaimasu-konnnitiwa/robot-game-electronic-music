# robot-game-electronic-music

ロボット・ゲーム・電気製品で使用する効果音素材です。
つくばチャレンジ2025において，屋外自律移動ロボットNexora-Redの効果音として利用されました.

言葉を持たないロボットや電化製品が、音楽的な響き（Vocalese）を用いて感情や意図を表現し、人間とコミュニケーションを行うための新しい「言語としての音楽」を創出し、社会で活用可能な技術へと発展させることを目的として、
「Non-verbal Robotic Musical Interaction」 という新たな研究分野の確立を目指している。

本分野では、従来の音声認識や会話型インタフェースに依存せず、ロボット固有の表現手段である音や効果音を活用することで、人間に対して直感的で分かりやすい意思伝達を実現する。その結果、言語能力を持たないロボットであっても、周囲の人々に安心感や親しみ、注意喚起、意思表示などを適切に伝えながら、円滑な相互関係を構築することが可能となる。
